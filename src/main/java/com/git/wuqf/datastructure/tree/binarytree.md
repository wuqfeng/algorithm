## 使用栈实现非递归遍历

任何递归函数都可以改成非递归。可以使用栈实现二叉树的非递归遍历。

### 先序遍历：

1. 先把头节点放到栈内
2. 弹出节点，弹出就打印
3. 如果有右孩子先压右孩子
4. 如果有左孩子压入左孩子

### 中序遍历

1. 整条左边界依次入栈
2. 当规则1无法继续，弹出节点，并打印
3. 在弹出节点的右树上执行规则1



依据原理：整棵树是可以被左边界分解掉的。

![image-20210520155551551](C:\Users\htwy\AppData\Roaming\Typora\typora-user-images\image-20210520155551551.png)





### 后序遍历

1. 左右树都没有处理，处理左树
2. 右树没有处理，处理右树
3. 左右两棵树都已经处理，打印并返回
